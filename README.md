# Веб-сервис координации волонтёров

Веб-приложение для координации волонтёров на городских мероприятиях и социальных проектах. Позволяет организаторам публиковать события и набирать добровольцев, а волонтёрам — находить подходящие мероприятия и записываться на участие.

---

## Требования к приложению

### Функциональные требования

#### Роли пользователей
- **Волонтёр** — регистрация, просмотр мероприятий, запись на события, управление своими заявками, просмотр истории участия.
- **Организатор** — создание и редактирование мероприятий, управление заявками волонтёров, назначение ролей и смен, коммуникация с волонтёрами.
- **Администратор** — управление пользователями, модерация мероприятий и контента, доступ к аналитике и отчётам.

#### Управление мероприятиями
- Создание мероприятия: название, описание, дата и время, место, категория, количество требуемых волонтёров, список ролей/задач.
- Редактирование и отмена мероприятий.
- Публикация и снятие с публикации.
- Фильтрация и поиск мероприятий по дате, категории, локации, ключевым словам.

#### Заявки и участие
- Подача заявки волонтёром на мероприятие (с указанием желаемой роли при необходимости).
- Подтверждение или отклонение заявок организатором.
- Назначение волонтёров на смены/зоны/задачи.
- Уведомления о статусе заявки и изменениях в мероприятии.

#### Профили и репутация
- Профиль волонтёра: контакты, навыки, интересы, история участия.
- Профиль организатора: информация об организации, список проведённых мероприятий.
- Учёт участия (часы, мероприятия) для волонтёрских книжек или отчётов.

#### Коммуникация
- Уведомления в приложении (о новых мероприятиях, смене статуса заявки, напоминаниях).
- При необходимости: общий чат мероприятия или связь с организатором.

#### Безопасность и доступ
- Регистрация и авторизация (логин/пароль, при необходимости восстановление пароля).
- Разграничение прав по ролям.
- Защита персональных данных и соответствие требованиям законодательства.

### Нефункциональные требования

- **Удобство (UX)**: понятная навигация, адаптивный интерфейс для мобильных устройств.
- **Производительность**: быстрая загрузка списков мероприятий и отклик интерфейса.
- **Надёжность**: сохранение данных заявок и корректная работа при сбоях сети (по возможности — офлайн-режим или очередь действий).
- **Масштабируемость**: возможность роста числа пользователей и мероприятий без потери скорости работы.

## Функционал приложения

### Для волонтёров

| Функция | Описание |
|--------|----------|
| Регистрация и вход | Создание учётной записи, вход в личный кабинет |
| Каталог мероприятий | Просмотр списка актуальных мероприятий с фильтрами и поиском |
| Карточка мероприятия | Подробное описание: дата, место, роли, организатор, кнопка «Участвовать» |
| Подача заявки | Запись на мероприятие, выбор роли/смены (если предусмотрено) |
| Мои заявки | Список поданных заявок со статусами: на рассмотрении, одобрена, отклонена |
| Календарь участия | Мероприятия, на которые волонтёр записан, с напоминаниями |
| Профиль | Редактирование контактов, навыков; просмотр истории участия и часов |
| Уведомления | Оповещения о статусе заявок и изменениях в мероприятиях |

### Для организаторов

| Функция | Описание |
|--------|----------|
| Создание мероприятия | Форма: название, описание, дата/время, место, категория, роли, лимит волонтёров |
| Редактирование мероприятия | Изменение данных, отмена или перенос события |
| Заявки волонтёров | Список заявок с подтверждением/отклонением, назначение на роли и смены |
| Управление участниками | Список одобренных волонтёров, контакты, рассылка напоминаний (если реализовано) |
| Мои мероприятия | Список созданных мероприятий (активные, завершённые, черновики) |
| Профиль организатора | Информация об организации, контакты для волонтёров |

### Для администраторов

| Функция | Описание |
|--------|----------|
| Модерация мероприятий | Проверка и публикация/отклонение мероприятий от организаторов |
| Управление пользователями | Просмотр пользователей, блокировка, назначение ролей |
| Аналитика | Статистика: количество мероприятий, волонтёров, заявок, активность по периодам |
| Настройки системы | Категории мероприятий, шаблоны уведомлений, общие настройки платформы |

### Общий функционал

- **Главная страница**: анонсы мероприятий, быстрый поиск, вход/регистрация.
- **Категории мероприятий**: культура, спорт, экология, социальная помощь, образование и др.
- **Поиск и фильтры**: по дате, категории, городу/району, ключевым словам.
- **Адаптивная вёрстка**: удобное использование с телефона и планшета.
- **Уведомления**: в интерфейсе (и при необходимости по email) о заявках и изменениях.

## Технологический стек

- **Frontend**: React 19, Vite 7
- **Сборка**: Vite
- **Стилизация**: CSS (при расширении — CSS-модули или иной подход по выбору)

Backend, база данных и авторизация в текущей версии не реализованы и подлежат проектированию (REST API, JWT/OAuth, PostgreSQL и т.п.) в рамках разработки дипломного проекта.
